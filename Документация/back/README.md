## Быстрый старт

1. Установи зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Создай файл `.env` на основе `.env.example`.

3. Запусти сервер:
   ```bash
   uvicorn back:app --reload --host 0.0.0.0 --port 3344
   ```

API будет доступен по адресу `http://127.0.0.1:3344`, а Swagger UI откроется на `http://127.0.0.1:3344/docs`.

## Особенности запуска

- При старте приложение автоматически создаёт таблицы.
- Если база пустая и `INIT_SAMPLE_DATA=true`, проект заполнит меню, отзывы, галерею и создаст администратора.
- CORS берётся из переменной `ALLOWED_ORIGINS`.

## Production

В Docker backend стартует той же командой `uvicorn back:app --host 0.0.0.0 --port 3344`, поэтому логика запуска совпадает с локальной и на хостинге.
